TEMPLATE for CH GUI (GTK) : template for prefixed "chg" apps. 
GUI apps based on chgtkt template use common gtkt.conf file. 
gtkt.conf file must be placed in /usr/local/bin/gtkt/ folder (or whole gtkt folder must be placed in /usr/local/bin/).  

(chgtkt must be recompiled with new chdaemon and nem function names).
